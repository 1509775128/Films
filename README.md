电影票售票系统的设计与实现
=====

本系统采用Apache License2.0开源协议。

系统说明
-----------
>1. 系统平台架构：B/S<br /> 
>2. Web服务器：Tomcat 7.0<br /> 
>3. 开发工具：eclipse<br /> 
>4. 数据库：MySQL5.5<br /> 
>5. 程序语言：java<br /> 
>6. 程序开发框架：Spring + Struts +Hibernate<br /> 
>7. 系统前台页面仿万达旧版网站页面，实现技术：jsp+javascript<br />
